# User Story Map

## Product: Cometogether

---

## Backbone (User Activities)

1. Onboarding  
2. Profile Management  
3. Discover Musicians  
4. Communication  
5. Band Management  
6. Rehearsal Organization (Future)  
7. Rehearsal Booking (Future)  

---

## 1. Onboarding

### User Stories
- As a user, I want to sign up using email or social login  
- As a user, I want to log in securely  
- As a user, I want a simple onboarding flow  

### MVP
✔ Sign up (email)  
✔ Login  
✔ Basic onboarding  

---

## 2. Profile Management

### User Stories
- As a user, I want to create my profile  
- As a user, I want to select my instrument(s)  
- As a user, I want to specify my skill level  
- As a user, I want to add preferred genres  
- As a user, I want to add a short bio  

### MVP
✔ Create/edit profile  
✔ Instrument  
✔ Skill level  
✔ Genres  

### Post-MVP
- Upload audio/video samples  
- Portfolio section  

---

## 3. Discover Musicians

### User Stories
- As a user, I want to browse musicians  
- As a user, I want to filter by instrument  
- As a user, I want to filter by genre  
- As a user, I want to filter by skill level  
- As a user, I want to search by location  

### MVP
✔ Basic search  
✔ Filter by instrument  
✔ Filter by genre  

### Post-MVP
- Advanced filtering  
- Matching algorithm  

---

## 4. Communication

### User Stories
- As a user, I want to send messages  
- As a user, I want to receive messages  
- As a user, I want notifications about new messages  

### MVP
✔ Basic messaging  

### Post-MVP
- Real-time chat  
- Notifications system  

---

## 5. Band Management

### User Stories
- As a user, I want to create a band  
- As a user, I want to invite musicians  
- As a user, I want to join a band  
- As a user, I want to manage band members  

### MVP
✔ Create band  
✔ Join band  

### Post-MVP
- Roles (admin/member)  
- Band activity feed  

---

## 6. Rehearsal Organization (Future)

### User Stories
- As a band member, I want to schedule rehearsals  
- As a band, I want to manage rehearsal times  

---

## 7. Rehearsal Booking (Future)

### User Stories
- As a musician, I want to find rehearsal spaces  
- As a musician, I want to book a rehearsal room  
- As a studio owner, I want to list my space  
- As a studio owner, I want to manage bookings  

---

## MVP Slice

The MVP will include the minimum features required to validate the core idea:

- User authentication  
- Profile creation  
- Musician discovery (basic filtering)  
- Messaging  
- Basic band creation  

---

## Release Plan (High-Level)

### MVP (Phase 1)
- Onboarding  
- Profiles  
- Discovery  
- Messaging  
- Band creation  

### Phase 2
- Advanced search  
- Notifications  
- Band management improvements  

### Phase 3
- Rehearsal organization  

### Phase 4
- Rehearsal space marketplace (booking system)
