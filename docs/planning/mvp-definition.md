# MVP Definition

## Product: Cometogether

---

## MVP Goal

Validate the core hypothesis:

Musicians are willing to use a dedicated platform to find bandmates,
communicate, and form bands.

---

## MVP Scope

The MVP will focus on the core value:

👉 Helping musicians find each other and start collaboration.

---

## Included Features (In Scope)

### 1. User Authentication
- Sign up with email  
- Login  
- Basic authentication flow  

---

### 2. User Profile

Users can create and manage a basic musician profile:

- Instrument(s)  
- Skill level  
- Preferred genres  
- Short bio  

---

### 3. Musician Discovery

Users can discover other musicians:

- Browse list of musicians  
- Filter by instrument  
- Filter by genre  

---

### 4. Communication

- Send messages to other users  
- Receive messages  

(Simple version, no real-time required)

---

### 5. Band Creation (Basic)

- Create a band  
- Join a band  

---

## Excluded Features (Out of Scope)

The following features are intentionally excluded from MVP:

### Advanced Features
- Advanced filtering and matching algorithms  
- Recommendation system  

### Profile Enhancements
- Audio/video uploads  
- Portfolio or media content  

### Communication Enhancements
- Real-time chat  
- Push/email notifications  

### Band Management
- Roles and permissions  
- Complex band management  

### Rehearsal Features
- Rehearsal scheduling  
- Rehearsal space booking  
- Studio accounts  

---

## Success Criteria

The MVP will be considered successful if:

- Users complete profiles  
- Users search and view other musicians  
- Users initiate conversations  
- Users form bands  

---

## Risks and Assumptions

### Assumptions
- Musicians are willing to create profiles  
- Users see value in a dedicated platform  
- Basic filtering is enough for early adoption  

### Risks
- Low user engagement  
- Lack of critical mass of users  
- Users prefer existing platforms (e.g. social media)  

---

## MVP Principles

- Keep it simple  
- Deliver fast  
- Focus on core value only  
- Avoid over-engineering  

---

## Out of MVP Vision (Future Phases)

- Advanced discovery and matching  
- Full-featured band management  
- Rehearsal organization tools  
- Rehearsal space marketplace and booking system
